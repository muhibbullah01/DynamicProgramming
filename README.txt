# Muhib1
There are some broken pieces of some scales. You need to merge the broken
pieces with glue and make it a large scale. But when you are making the
large scale, you will select the minimum number of broken pieces. Print the
selected broken pieces length to make the large scale.
Broken pieces length: 1, 2, 3, 4, 5, 6, 7, 8
Large Scale length: 14
